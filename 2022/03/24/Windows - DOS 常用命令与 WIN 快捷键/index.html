Windows - DOS 常用命令与 WIN 快捷键

dos常用命令

  • 回到根目录

  • 清屏

  • 查看当前目录下的文件

  • 退出DOS命令窗口

  • 自动补全：Tab键

  • 删除文件或目录下所有文件夹

  • 删除文件夹

  • 查看ip地址

  • 测试连通性

  • 终止无限命令：Ctrl + C

  • 拖动文件到dos窗口，会自动生成文件的绝对路径

  • 文件夹目录地址输入cmd，会进入到目录的命令窗口

win快捷键

  • 回到文件头：Ctrl + Home
  • 回到文件尾：Ctrl + End
  • 选中一个单词
    • 鼠标：双击
    • 键盘：Ctrl + Shift + Left/Right
  • 选中一行：
    • 鼠标：三击
    • 键盘：Shift + Home/End
