# dos常用命令

- 回到根目录

    ```
    cd /
    ```

- 清屏

    ```
    cls
    ```

- 查看当前目录下的文件

    ```
    dir
    ```

- 退出DOS命令窗口

    ```
    exit
    ```

- 自动补全：`Tab键`

- 删除文件或目录下所有文件夹

  ```
  del
  del *[name]	// 删除匹配到的文件
  ```

- 删除文件夹

  ```
  rd
  ```

- 查看ip地址

  ```
  ipconfig
  ipconfig /all	// 查看详细信息
  ipconfig/flushdns	// 刷新dns缓存
  ```

- 测试连通性

  ```
  ping [网址/域名]
  ping [网址/域名] -t	// 连续测试
  ```

- 终止无限命令：`Ctrl + C`

- `拖动文件到dos窗口`，会自动生成文件的绝对路径

- `文件夹目录地址输入cmd`，会进入到目录的命令窗口

# win快捷键

- 回到文件头：`Ctrl + Home`
- 回到文件尾：`Ctrl + End`
- 选中一个单词
  - 鼠标：`双击`
  - 键盘：`Ctrl + Shift + Left/Right`
- 选中一行：
  - 鼠标：`三击`
  - 键盘：`Shift + Home/End`
